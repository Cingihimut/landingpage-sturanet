# Core Concepts
With blockchain, every transaction is processed through a cryptographic hash, making it transparent and verifiable by anyone. This technology ensures all transactions are publicly recorded and cannot be altered. This provides a solution to the trust and transparency issues in traditional centralized crowdfunding platforms. Additionally, the use of digital currency in this system allows contributors from different countries to participate without currency conversion constraints.

## solution
Sturan Network also adopts smart contracts to automate the transaction process based on certain conditions, improving efficiency and security. The system provides open transaction records, independent verification, and ensures that funds are sent to authorized recipients without manipulation.
